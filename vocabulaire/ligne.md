# Ligné - מְשֻׁרְטָט (meshourtate)

## Information principale

**Mot hébreu :** מְשֻׁרְטָט (meshourtate) - masculin
**Féminin :** מְשֻׁרְטֶטֶת (meshourtétét)
**Pluriel masculin :** מְשֻׁרְטָטִים (meshourtațim)
**Pluriel féminin :** מְשֻׁרְטָטוֹת (meshourtațot)

**Racine (שֹׁרֶשׁ) :** ש-ר-ט (shin-resh-tet)
**Binyan :** פּוּעַל (Pu'al) - forme passive

**Étymologie :** Le mot vient de la racine שׁ-ר-ט qui signifie "graver, tracer, rayer". Le binyan Pu'al indique qu'il s'agit d'une forme passive/adjectivale, signifiant "qui a été ligné/rayé".

## Accord grammatical

Comme tous les adjectifs en hébreu, מְשֻׁרְטָט doit s'accorder en genre et en nombre avec le nom qu'il qualifie :

| Genre/Nombre | Forme hébraïque | Phonétique | Exemple |
|--------------|----------------|------------|---------|
| Masculin singulier | מְשֻׁרְטָט | meshourtate | פִּנְקָס מְשֻׁרְטָט (pinkas meshourtate) - bloc-notes ligné |
| Féminin singulier | מְשֻׁרְטֶטֶת | meshourtétét | מַחְבֶּרֶת מְשֻׁרְטֶטֶת (makhbérét meshourtétét) - cahier ligné |
| Masculin pluriel | מְשֻׁרְטָטִים | meshourtațim | דַּפִּים מְשֻׁרְטָטִים (dapim meshourtațim) - feuilles lignées |
| Féminin pluriel | מְשֻׁרְטָטוֹת | meshourtațot | מַחְבָּרוֹת מְשֻׁרְטָטוֹת (makhbarot meshourtațot) - cahiers lignés |

**Note importante :** מַחְבֶּרֶת (cahier) est un nom féminin, donc on utilise la forme féminine מְשֻׁרְטֶטֶת.

## Exemples de phrases

### Avec "cahier ligné"

1. **אֲנִי קוֹנֶה מַחְבֶּרֶת מְשֻׁרְטֶטֶת לְבֵית הַסֵּפֶר.**
   - *Ani koneh makhbérét meshourtétét lebéyt haséfèr.*
   - Je achète un cahier ligné pour l'école.

2. **הַמַּחְבֶּרֶת הַמְּשֻׁרְטֶטֶת עוֹזֶרֶת לִכְתֹּב בְּיֹפִי.**
   - *Hamakhbérét hameshourtétét ozérét likhtov beyofi.*
   - Le cahier ligné aide à écrire joliment.

3. **הִיא כּוֹתֶבֶת רַק בְּמַחְבָּרוֹת מְשֻׁרְטָטוֹת.**
   - *Hi kotévét rak bemakhbarot meshourtațot.*
   - Elle écrit seulement dans des cahiers lignés.

4. **הַיְלָדִים צְרִיכִים מַחְבֶּרֶת מְשֻׁרְטֶטֶת לְשִׁעוּר עִבְרִית.**
   - *Hayeladim țerikhim makhbérét meshourtétét leshi'our ivrit.*
   - Les enfants ont besoin d'un cahier ligné pour le cours d'hébreu.

### Avec d'autres objets lignés

5. **הוּא מֵעֲדִיף נְיָר מְשֻׁרְטָט.**
   - *Hou mé'adif neyar meshourtate.*
   - Il préfère du papier ligné.

6. **יֵשׁ לִי פִּנְקָס מְשֻׁרְטָט בְּתִיק.**
   - *Yésh li pinkas meshourtate batik.*
   - J'ai un bloc-notes ligné dans le sac.

7. **הַדַּף הַמְּשֻׁרְטָט עוֹזֵר לִי לִכְתֹּב יָשָׁר.**
   - *Hadaf hameshourtate ozér li likhtov yashar.*
   - La feuille lignée m'aide à écrire droit.

## Vocabulaire connexe

### Les lignes et le lignage

1. **שׁוּרָה** (shourạ́) - ligne, rangée (féminin)
   - *Exemple : כָּל שׁוּרָה בַּמַּחְבֶּרֶת יְשָׁרָה.* (Kol shourạ́ bamakhbérét yesharạ́.) - Chaque ligne dans le cahier est droite.

2. **קַו** (kav) - ligne, trait (masculin)
   - *Exemple : הַקַּו הָאֲדֹם מַפְרִיד בֵּין הַשּׁוּרוֹת.* (Hakav ha'adóm mafrịd béyn hashhourot.) - La ligne rouge sépare entre les lignes.

3. **לְשַׁרְטֵט** (lesharțéț) - tracer, dessiner (verbe - binyan Pi'el)
   - *Racine : ש-ר-ט*
   - *Exemple : אֲנִי מְשַׁרְטֵט קַו בַּמַּחְבֶּרֶת.* (Ani mesharțéț kav bamakhbérét.) - Je trace une ligne dans le cahier.

4. **שִׁרְטוּט** (shirțout) - tracé, dessin technique (masculin)
   - *Exemple : זֶה מַחְבֶּרֶת לְשִׁרְטוּט.* (Zeh makhbérét leshirțout.) - C'est un cahier pour le dessin technique.

### Types de lignes

1. **קַוִּים אֲפֻקִּיִּים** (kavịm afoukiyịm) - lignes horizontales
   - *Exemple : בַּמַּחְבֶּרֶת הַמְּשֻׁרְטֶטֶת יֵשׁ קַוִּים אֲפֻקִּיִּים.* (Bamakhbérét hameshourtétét yésh kavịm afoukiyịm.) - Dans le cahier ligné il y a des lignes horizontales.

2. **קַוִּים אֲנָכִיִּים** (kavịm anakhiyịm) - lignes verticales
   - *Exemple : הַטַּבְלָה כּוֹלֶלֶת קַוִּים אֲנָכִיִּים.* (Hațavlạ́ kolélét kavịm anakhiyịm.) - Le tableau contient des lignes verticales.

3. **רֶשֶׁת / סְרִיג** (réshét / serig) - grille, quadrillage
   - *Exemple : מַחְבֶּרֶת לְמָתֶמָטִיקָה עִם סְרִיג.* (Makhbérét lemațemațika im serig.) - Cahier de mathématiques avec quadrillage.

### Autres descripteurs de cahiers

1. **מְרֻבָּע / מְרֻבַּעַת** (meroubá' / merouba'át) - quadrillé(e)
   - **מַחְבֶּרֶת מְרֻבַּעַת** (makhbérét merouba'át) - cahier quadrillé
   - *Exemple : לְמָתֶמָטִיקָה אֲנִי צָרִיךְ מַחְבֶּרֶת מְרֻבַּעַת.* (Lemațemațika ani țarikh makhbérét merouba'át.) - Pour les mathématiques j'ai besoin d'un cahier quadrillé.

2. **חָלָק / חֲלָקָה** (khalák / khalakạ́) - lisse, vierge, blanc
   - **מַחְבֶּרֶת חֲלָקָה** (makhbérét khalakạ́) - cahier vierge/blanc
   - *Exemple : לְצִיּוּר אֲנִי מֵעֲדִיף מַחְבֶּרֶת חֲלָקָה.* (Letsiýour ani mé'adif makhbérét khalakạ́.) - Pour le dessin je préfère un cahier vierge.

3. **רָגִיל / רְגִילָה** (ragịl / regilạ́) - normal, ordinaire
   - **מַחְבֶּרֶת רְגִילָה** (makhbérét regilạ́) - cahier ordinaire

4. **סָפִיר** (sapir) - spiral (cahier à spirale)
   - **מַחְבֶּרֶת סָפִיר** (makhbérét sapir) - cahier spiralé
   - *Exemple : מַחְבֶּרֶת סָפִיר מְשֻׁרְטֶטֶת.* (Makhbérét sapir meshourtétét.) - Cahier spiralé ligné.

5. **קָשִׁיחַ / קְשִׁיחָה** (kashịakh / keshikhạ́) - rigide, à couverture dure
   - **מַחְבֶּרֶת קְשִׁיחָה** (makhbérét keshikhạ́) - cahier à couverture rigide
