# dans - בְּ (be)

**Type :** Préposition préfixée (s'attache au mot suivant)
**Phonétique :** be

## Explication grammaticale complète

En hébreu, "dans" se traduit principalement par la préposition **בְּ** (be), qui est une **préposition préfixée** : elle s'attache directement au mot qui suit et ne s'écrit jamais séparément.

### Règles de vocalisation de בְּ :

1. **בְּ** (be) avec shva : devant la plupart des consonnes
   - Exemple : בְּבַיִת (be-bayit) = dans une maison

2. **בַּ** (ba) avec patah : devant les lettres à shva (ב, כ, פ, ג, ד, ת avec shva)
   - Exemple : בַּכִּתָה (ba-kita) = dans la classe

3. **בָּ** (ba) avec qamats : devant l'article défini הַ (ha)
   - הַבַּיִת (ha-bayit) = la maison → בַּבַּיִת (ba-bayit) = dans la maison
   - הַסֵּפֶר (ha-sefer) = le livre → בַּסֵּפֶר (ba-sefer) = dans le livre

4. **בִּ** (bi) avec hiriq : devant certains mots et dans des expressions figées
   - בִּירוּשָׁלַיִם (bi-Yeroushalayim) = à/dans Jérusalem

### Autres expressions pour "dans" :

- **בְּתוֹךְ** (betokh) : à l'intérieur de, au milieu de (plus explicite)
- **לְתוֹךְ** (letokh) : vers l'intérieur de (avec mouvement)

## Exemples de phrases (10 phrases)

### 1. יֵשׁ סֵפֶר בַּתִּיק
**Phonétique :** yesh sefer ba-tik
**Français :** Il y a un livre dans le sac

### 2. הַיֶּלֶד יוֹשֵׁב בַּגַּן
**Phonétique :** ha-yeled yoshev ba-gan
**Français :** L'enfant est assis dans le jardin

### 3. אֲנִי גָּר בְּתֵל אָבִיב
**Phonétique :** ani gar be-Tel Aviv
**Français :** J'habite à Tel Aviv (dans Tel Aviv)

### 4. הַמַּפְתֵּחַ נִמְצָא בַּכִּיס
**Phonétique :** ha-maftéakh nimtsa ba-kis
**Français :** La clé se trouve dans la poche

### 5. נִפְגַּשׁ בְּעוֹד שָׁעָה
**Phonétique :** nifgash be-od sha'a
**Français :** On se rencontre dans une heure

### 6. הַחַתּוּל יָשֵׁן בַּקּוּפְסָה
**Phonétique :** ha-khatoul yashen ba-koufsah
**Français :** Le chat dort dans la boîte

### 7. יֵשׁ הַרְבֵּה אֲנָשִׁים בָּרְחוֹב
**Phonétique :** yesh harbeh anashim ba-rekhov
**Français :** Il y a beaucoup de gens dans la rue

### 8. אֲנַחְנוּ אוֹכְלִים בַּמִּסְעָדָה
**Phonétique :** anakhnou okhlim ba-mis'adah
**Français :** Nous mangeons dans le restaurant

### 9. הַמּוֹרָה כּוֹתֶבֶת עַל הַלּוּחַ בַּכִּתָּה
**Phonétique :** ha-morah kotevet al ha-louakh ba-kitah
**Français :** La professeure écrit au tableau dans la classe

### 10. בְּתוֹךְ הַסֵּפֶר יֵשׁ תְּמוּנוֹת יָפוֹת
**Phonétique :** betokh ha-sefer yesh tmounot yafot
**Français :** Dans le livre il y a de belles images

## Proposition de carte Anki

**RECTO :**
dans (préposition de lieu)

**VERSO :**
**בְּ** (be)
Préposition préfixée qui s'attache au mot suivant

Exemples :
- בַּבַּיִת (ba-bayit) = dans la maison
- בַּגַּן (ba-gan) = dans le jardin
- בַּסֵּפֶר (ba-sefer) = dans le livre

## Corrigés des exercices

### Section A : Lieux physiques

**1.** הַחַתּוּל נִמְצָא בַּבַּיִת
**Phonétique :** ha-khatoul nimtsa ba-bayit
**Explication :** בַּבַּיִת = dans la maison (בְּ + הַבַּיִת → בַּבַּיִת)

**2.** יֵשׁ פְּרָחִים בַּגַּן
**Phonétique :** yesh prakhim ba-gan
**Explication :** בַּגַּן = dans le jardin (בְּ + הַגַּן → בַּגַּן)

**3.** הַיְלָדִים מְשַׂחֲקִים בַּפַּרְק
**Phonétique :** ha-yeladim mesakhaqim ba-park
**Explication :** בַּפַּרְק = dans le parc (בְּ + הַפַּרְק → בַּפַּרְק)

**4.** אֲנַחְנוּ גָּרִים בְּעִיר גְּדוֹלָה
**Phonétique :** anakhnou garim be-ir gdolah
**Explication :** בְּעִיר = dans une ville (pas d'article défini)

**5.** L'eau dans le verre est froide
**Hébreu :** הַמַּיִם בַּכּוֹס קָרִים
**Phonétique :** ha-mayim ba-kos karim
**Explication :** בַּכּוֹס = dans le verre

**6.** Le bébé dort dans le lit
**Hébreu :** הַתִּינוֹק יָשֵׁן בַּמִּטָּה
**Phonétique :** ha-tinok yashen ba-mitah
**Explication :** בַּמִּטָּה = dans le lit

**7.** הַסֵּפֶר נִמְצָא בַּמַּגְרֵרָה
**Phonétique :** ha-sefer nimtsa ba-magrerah
**Explication :** בַּמַּגְרֵרָה = dans le tiroir

**8.** הִיא עוֹבֶדֶת בְּמִשְׂרָד
**Phonétique :** hi ovedet be-misrad
**Explication :** בְּמִשְׂרָד = dans un bureau

### Section B : Objets et contenants

**9.** יֵשׁ קָפֶה בַּכּוֹס
**Phonétique :** yesh kafeh ba-kos
**Explication :** בַּכּוֹס = dans la tasse

**10.** Le chien mange dans la gamelle
**Hébreu :** הַכֶּלֶב אוֹכֵל בַּקְּעָרָה
**Phonétique :** ha-kelev okhel ba-ke'arah
**Explication :** בַּקְּעָרָה = dans la gamelle/le bol

**11.** הַבְּגָדִים נִמְצָאִים בָּאָרוֹן
**Phonétique :** ha-bgadim nimtsa'im ba-aron
**Explication :** בָּאָרוֹן = dans l'armoire

**12.** אֲנִי כּוֹתֵב בַּמַּחְבֶּרֶת שֶׁלִּי
**Phonétique :** ani kotev ba-makhberet sheli
**Explication :** בַּמַּחְבֶּרֶת = dans le cahier

**13.** Le téléphone se trouve dans le sac
**Hébreu :** הַטֶּלֶפוֹן נִמְצָא בַּתִּיק
**Phonétique :** ha-telefon nimtsa ba-tik
**Explication :** בַּתִּיק = dans le sac

**14.** יֵשׁ כֶּסֶף בָּאַרְנָק
**Phonétique :** yesh kesef ba-arnak
**Explication :** בָּאַרְנָק = dans le portefeuille

### Section C : Expressions temporelles

**15.** אֲנִי נוֹסֵעַ בְּעוֹד יוֹמַיִם
**Phonétique :** ani noséa be-od yomayim
**Explication :** בְּעוֹד יוֹמַיִם = dans deux jours (littéralement : "dans encore deux jours")

**16.** הִיא מַגִּיעָה בְּעוֹד שָׁעָה
**Phonétique :** hi magia be-od sha'ah
**Explication :** בְּעוֹד שָׁעָה = dans une heure

**17.** On te voit dans une semaine
**Hébreu :** נִרְאֶה אוֹתְךָ בְּעוֹד שָׁבוּעַ
**Phonétique :** nireh otkha be-od shavouа
**Explication :** בְּעוֹד שָׁבוּעַ = dans une semaine

**18.** אֲנַחְנוּ נִפְגָּשִׁים בְּעוֹד עֶשֶׂר דַּקּוֹת
**Phonétique :** anakhnou nifgashim be-od eser dakot
**Explication :** בְּעוֹד עֶשֶׂר דַּקּוֹת = dans dix minutes

**19.** L'examen sera dans un mois
**Hébreu :** הַמִּבְחָן יִהְיֶה בְּעוֹד חֹדֶשׁ
**Phonétique :** ha-mivkhan yihyeh be-od khodesh
**Explication :** בְּעוֹד חֹדֶשׁ = dans un mois

**20.** הַמְּסִבָּה מַתְחִילָה בְּעוֹד שָׁלוֹשׁ שָׁעוֹת
**Phonétique :** ha-mesibah matkhilah be-od shalosh sha'ot
**Explication :** בְּעוֹד שָׁלוֹשׁ שָׁעוֹת = dans trois heures
