# soleil - שֶׁמֶשׁ (shemesh)

**Genre :** Féminin
**Nombre :** Singulier
**Pluriel :** שְׁמָשׁוֹת (shmashot)

## Exemples de phrases

1. **הַשֶּׁמֶשׁ זוֹרַחַת בַּשָּׁמַיִם**
   - *Ha-shemesh zorakhat ba-shamayim*
   - Le soleil brille dans le ciel

2. **הַיּוֹם הַשֶּׁמֶשׁ חַמָּה מְאֹד**
   - *Ha-yom ha-shemesh khama me'od*
   - Aujourd'hui le soleil est très chaud

3. **אֲנִי אוֹהֵב לִרְאוֹת אֶת הַשֶּׁמֶשׁ**
   - *Ani ohev lir'ot et ha-shemesh*
   - J'aime voir le soleil

4. **הַשֶּׁמֶשׁ שׁוֹקַעַת בָּעֶרֶב**
   - *Ha-shemesh shoka'at ba-erev*
   - Le soleil se couche le soir

## Corrigés des exercices

1. **הַשֶּׁמֶשׁ יָפָה**
   - *Ha-shemesh yafa*
   - Le soleil est beau
   - Explication : L'adjectif יָפָה (yafa) est au féminin pour s'accorder avec שֶׁמֶשׁ qui est un nom féminin.

2. **הַשֶּׁמֶשׁ גְּדוֹלָה**
   - *Ha-shemesh gdola*
   - Le soleil est grand
   - Explication : L'adjectif גְּדוֹלָה (gdola) est au féminin pour s'accorder avec שֶׁמֶשׁ.

3. **הַיּוֹם הַשֶּׁמֶשׁ חַמָּה**
   - *Ha-yom ha-shemesh khama*
   - Aujourd'hui le soleil est chaud
   - Explication : הַשֶּׁמֶשׁ (ha-shemesh) est le mot manquant.

4. **אֲנִי אוֹהֵב/אוֹהֶבֶת אֶת הַשֶּׁמֶשׁ**
   - *Ani ohev (masculin) / ohevet (féminin) et ha-shemesh*
   - J'aime le soleil
   - Explication : Le verbe אוֹהֵב/אוֹהֶבֶת (aimer) s'accorde selon le genre du locuteur. La particule אֶת (et) introduit le complément d'objet direct défini.

5. **שְׁמָשׁוֹת** (shmashot)
   - Soleils
   - Explication : Le pluriel de שֶׁמֶשׁ est שְׁמָשׁוֹת, suivant le modèle classique des noms féminins segolés.
