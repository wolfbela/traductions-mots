# se situer - לְהִמָּצֵא / לְהִתְמַקֵּם

## Vue d'ensemble

Le verbe français "se situer" se traduit en hébreu par plusieurs verbes selon le contexte :

1. **לְהִמָּצֵא** (lehimatsé) - être situé, se trouver [PRINCIPAL - état passif]
2. **לְהִתְמַקֵּם** (lehitmakem) - se positionner, s'installer [action active]
3. **לָשֶׁבֶת** (lashévet) - être assis/situé [permanence]

---

## 1. לְהִמָּצֵא (lehimatsé) - "être situé/se trouver" [PRINCIPAL]

**Racine (שֹׁרֶשׁ) :** מ-צ-א (m-ts-a)
**Binyan :** נִפְעַל (Nif'al) - forme passive/réfléchie
**Infinitif :** לְהִמָּצֵא (lehimatsé)
**Phonétique :** leh-hi-ma-TSÉ
**Traduction :** être situé, se trouver, être localisé

> **Note :** C'est le verbe le plus courant pour exprimer une localisation géographique ou physique.

### Exemples de phrases

#### Type 1 - Avec infinitif

1. **הַבַּיִת יָכוֹל לְהִמָּצֵא בְּמֶרְכַּז הָעִיר.**
   *ha-bá-yit ya-KHOL leh-hi-ma-TSÉ be-mer-KAZ ha-ÍR*
   "La maison peut se situer au centre de la ville."

2. **טוֹב לְהִמָּצֵא קָרוֹב לַתַּחֲנָה.**
   *tov leh-hi-ma-TSÉ ka-RÓV la-ta-kha-NÁ*
   "C'est bien d'être situé près de la gare."

#### Type 2 - Avec conjugaisons

**PRÉSENT :**

1. **הַמּוּזֵיאוֹן נִמְצָא בִּירוּשָׁלַיִם.**
   *ha-mou-zé-ON nim-TSÁ bi-rou-sha-LÁ-yim*
   "Le musée se situe à Jérusalem."

2. **אֲנַחְנוּ נִמְצָאִים בַּפַּרְק.**
   *a-nákh-nou nim-tsa-ÍM ba-PARK*
   "Nous nous trouvons dans le parc."

3. **הַמִּסְעָדָה נִמְצֵאת בָּרְחוֹב הָרִאשׁוֹן.**
   *ha-mis-a-DÁ nim-TSÉT ba-re-KHÓV ha-ri-SHÓN*
   "Le restaurant se situe dans la première rue."

**PASSÉ :**

4. **הַסֵּפֶר נִמְצָא עַל הַשֻּׁלְחָן אֶתְמוֹל.**
   *ha-SÉ-fer nim-TSÁ al ha-shoul-KHÁN et-MÓL*
   "Le livre se trouvait sur la table hier."

5. **הֵם נִמְצְאוּ בַּמַּצָּב קָשֶׁה.**
   *hem nim-tse-ÓU ba-ma-TSÁV ka-SHÉH*
   "Ils se sont trouvés dans une situation difficile."

**FUTUR :**

6. **הַבִּנְיָן יִמָּצֵא כָּאן בְּעוֹד שָׁנָה.**
   *ha-bin-YÁN yi-ma-TSÉ kan be-ÓD sha-NÁ*
   "Le bâtiment se situera ici dans un an."

7. **אַתֶּם תִּמָּצְאוּ בַּמָּקוֹם הַנָּכוֹן.**
   *a-TEM ti-ma-tse-ÓU ba-ma-KÓM ha-na-KHÓN*
   "Vous vous trouverez au bon endroit."

## 2. לְהִתְמַקֵּם (lehitmakem) - "se positionner/s'installer"

**Racine (שֹׁרֶשׁ) :** מ-ק-ם (m-k-m)
**Binyan :** הִתְפַּעֵל (Hitpa'el) - forme réfléchie
**Infinitif :** לְהִתְמַקֵּם (lehitmakem)
**Phonétique :** leh-hit-ma-KÉM
**Traduction :** se positionner, s'installer, se situer (activement)

> **Note :** Ce verbe implique une action active de se positionner, plutôt qu'un simple état de localisation.

   "Elle s'est installée dans le fauteuil confortable."

   *ha-maf-ÁL yit-ma-KÉM ba-e-ZÓR ha-ta-a-si-ya-TÍ*
   "L'usine se positionnera dans la zone industrielle."

## 3. לָשֶׁבֶת (lashévet) - "être assis/situé"

**Racine (שֹׁרֶשׁ) :** י-ש-ב (y-sh-b)
**Binyan :** פָּעַל (Pa'al) - forme de base
**Infinitif :** לָשֶׁבֶת (lashévet)
**Phonétique :** la-SHÉ-vet
**Traduction :** s'asseoir, habiter, être situé

> **Note :** Ce verbe peut signifier "être situé" dans un sens de permanence ou d'habitation.

### Conjugaison au présent (הֹוֶה)

| Personne | Hébreu | Phonétique | Français |
|----------|--------|------------|----------|
| אֲנִי/אַתָּה/הוּא (m.sg) | יוֹשֵׁב | yoSHÉV | je suis assis / il est assis |
| אַתְּ/הִיא (f.sg) | יוֹשֶׁבֶת | yoSHÉvet | tu es assise / elle est assise |
| אֲנַחְנוּ (nous) | יוֹשְׁבִים (m) / יוֹשְׁבוֹת (f) | yosh-VÍM / yosh-VÓT | nous sommes assis/assises |
| אַתֶּם (vous m) | יוֹשְׁבִים | yosh-VÍM | vous êtes assis |

| הֵן (elles) | יוֹשְׁבוֹת | yosh-VÓT | elles sont assises |

### Exemple d'usage pour "se situer"

**הַכְּפָר יוֹשֵׁב בֵּין הֶהָרִים.**
*ha-kfar yo-SHÉV ben he-ha-RÍM*
"Le village est situé entre les montagnes."

## Expressions courantes

1. **אֵיפֹה זֶה נִמְצָא?**
   *é-FO zé nim-TSÁ?*
   "Où est-ce que ça se trouve ?"

2. **הַמָּקוֹם נִמְצָא בְּמֶרְכַּז הָעִיר.**
   *ha-ma-KÓM nim-TSÁ be-mer-KÁZ ha-ÍR*
   "L'endroit se situe au centre-ville."

3. **אֲנִי נִמְצָא בְּמַצָּב טוֹב.**
   *a-NÍ nim-TSÁ be-ma-TSÁV tov*
   "Je me trouve dans une bonne situation."

4. **הַבַּיִת נִמְצָא בֵּין שְׁנֵי רְחוֹבוֹת.**
   *ha-BÁ-yit nim-TSÁ ben she-NÉ re-kho-VÓT*
   "La maison est située entre deux rues."

5. **הַחֲנוּת מִתְמַקֶּמֶת בָּאֵזוֹר הַמִּסְחָרִי.**
   *ha-kha-NÚT mit-ma-KÉ-met ba-e-ZÓR ha-mis-kha-RÍ*
   "Le magasin se positionne dans la zone commerciale."

6. **לְהִתְמַקֵּם בְּרֹאשׁ הָרְשִׁימָה**
   *le-hit-ma-KÉM be-ROSH ha-re-shi-MÁ*
   "Se positionner en tête de liste"

## Nuances contextuelles

### Quand utiliser לְהִמָּצֵא (lehimatsé) :
- Localisation géographique : "Le musée se situe..."
- État passif : "se trouver dans une situation"
- Position physique statique

### Quand utiliser לְהִתְמַקֵּם (lehitmakem) :
- Action active de se positionner
- Installation dans un lieu
- Positionnement stratégique (commercial, militaire, professionnel)

### Quand utiliser לָשֶׁבֶת (lashévet) :
- Situation permanente (village, bâtiment ancien)
- Sens d'habitation ou résidence

Binyan : נִפְעַל (Nif'al)
Racine : מ-צ-א

**Présent masculin singulier :** נִמְצָא (nimtsá)
**Exemple :** הַמּוּזֵיאוֹן נִמְצָא בִּירוּשָׁלַיִם
*(ha-mou-zé-ON nim-TSÁ bi-rou-sha-LÁ-yim)*
"Le musée se situe à Jérusalem"

---

### Carte 2 - לְהִתְמַקֵּם

**RECTO :**
Se positionner / s'installer (actif)
_Action volontaire de se placer quelque part_

**VERSO :**
לְהִתְמַקֵּם (lehitmakem)
Binyan : הִתְפַּעֵל (Hitpa'el)
Racine : מ-ק-ם

**Présent masculin singulier :** מִתְמַקֵּם (mitmakem)
**Exemple :** הַחֶבְרָה מִתְמַקֶּמֶת בַּשּׁוּק הַיִּשְׂרְאֵלִי
*(ha-khev-RÁ mit-ma-KÉ-met ba-SHOUK ha-yis-re-É-li)*
"L'entreprise se positionne sur le marché israélien"

---

### Carte 3 - Expression courante

**RECTO :**
Où est-ce que ça se trouve ?

**VERSO :**
אֵיפֹה זֶה נִמְצָא?
*(é-FO zé nim-TSÁ?)*

## Exercices de pratique

### Exercice 1 - Traduction FR → HE
Traduisez en hébreu avec nekoudot :

1. Le restaurant se trouve dans le centre-ville.
2. Nous nous positionnons près de la porte.
3. Où se situe la bibliothèque ?
4. L'école était située à côté du parc.
5. Je vais me positionner à gauche.

### Exercice 2 - Traduction HE → FR
Traduisez en français :

1. הַסֵּפֶר נִמְצָא עַל הַמַּדָּף.
2. הֵם מִתְמַקְּמִים בַּכִּתָּה.
3. הָעִיר נִמְצֵאת בַּצָּפוֹן.
4. אֲנַחְנוּ נִמָּצֵא כָּאן מָחָר.
5. הַמִּשְׂרָד הִתְמַקֵּם בְּבִנְיָן חָדָשׁ.

### Exercice 3 - Conjugaison
Conjuguez לְהִמָּצֵא (lehimatsé) :

1. (Présent, elle) ________ בַּבַּיִת.
2. (Passé, ils) ________ בָּאוּטוֹבּוּס.
3. (Futur, nous) ________ שָׁם.
4. (Présent, je - masculin) ________ בַּמַּטְבָּח.
5. (Passé, tu - féminin) ________ בָּרְחוֹב.

### Exercice 4 - Choix du verbe
Choisissez entre לְהִמָּצֵא et לְהִתְמַקֵּם :

1. Le musée ________ (se trouve) à Tel Aviv. [état passif]
2. L'armée ________ (se positionne) à la frontière. [action active]
3. Nous ________ (nous trouvons) dans une situation difficile. [état]
4. La compagnie ________ (se positionne) sur le marché. [action stratégique]
5. Où ________ (se trouve) la gare ? [localisation]

### Exercice 5 - Complétez les phrases
Complétez avec la forme correcte :

1. הַבַּיִת ________ (se trouve - présent) בֵּין שְׁנֵי רְחוֹבוֹת.
2. אֲנַחְנוּ ________ (nous nous positionnons - passé) בַּמָּקוֹם הַטּוֹב.
3. הַחֲנוּת ________ (se trouvera - futur) קָרוֹב לַתַּחֲנָה.
4. הֵם ________ (se sont positionnés - passé) בָּרֹאשׁ.
5. אֵיפֹה אַתְּ ________ (tu te trouves - présent, féminin)?

## Corrigés des exercices

### Corrigé Exercice 1

1. **הַמִּסְעָדָה נִמְצֵאת בְּמֶרְכַּז הָעִיר.**
   *(ha-mis-a-DÁ nim-TSÉT be-mer-KAZ ha-ÍR)*

2. **אֲנַחְנוּ מִתְמַקְּמִים קָרוֹב לַדֶּלֶת.**
   *(a-nákh-nou mit-mak-MÍM ka-RÓV la-DÉ-let)*

3. **אֵיפֹה נִמְצֵאת הַסִּפְרִיָּה?**
   *(é-FO nim-TSÉT ha-sif-ri-YÁ?)*

4. **בֵּית הַסֵּפֶר נִמְצָא בְּצַד הַפַּרְק.**
   *(bet ha-SÉ-fer nim-TSÁ be-TSAD ha-PARK)*

5. **אֲנִי אֶתְמַקֵּם בִּשְׂמֹאל.**
   *(a-NÍ et-ma-KÉM bis-MÓL)*

### Corrigé Exercice 2

1. **הַסֵּפֶר נִמְצָא עַל הַמַּדָּף.**
   *(ha-SÉ-fer nim-TSÁ al ha-ma-DÁF)*
   "Le livre se trouve sur l'étagère."

2. **הֵם מִתְמַקְּמִים בַּכִּתָּה.**
   *(hem mit-mak-MÍM ba-ki-TÁ)*
   "Ils se positionnent dans la classe."

3. **הָעִיר נִמְצֵאת בַּצָּפוֹן.**
   *(ha-ÍR nim-TSÉT ba-tsa-FÓN)*
   "La ville se situe au nord."

4. **אֲנַחְנוּ נִמָּצֵא כָּאן מָחָר.**
   *(a-nákh-nou ni-ma-TSÉ kan ma-KHÁR)*
   "Nous nous trouverons ici demain."

5. **הַמִּשְׂרָד הִתְמַקֵּם בְּבִנְיָן חָדָשׁ.**
   *(ha-mis-RÁD hit-ma-KÉM be-bin-YÁN kha-DÁSH)*
   "Le bureau s'est installé dans un nouveau bâtiment."

### Corrigé Exercice 3

1. **(Présent, elle)** נִמְצֵאת **בַּבַּיִת.**
   *(nim-TSÉT ba-BÁ-yit)*
   "Elle se trouve à la maison."

2. **(Passé, ils)** נִמְצְאוּ **בָּאוּטוֹבּוּס.**
   *(nim-tse-ÓU ba-o-to-BOUS)*
   "Ils se trouvaient dans le bus."

3. **(Futur, nous)** נִמָּצֵא **שָׁם.**
   *(ni-ma-TSÉ sham)*
   "Nous nous trouverons là-bas."

4. **(Présent, je - masculin)** נִמְצָא **בַּמַּטְבָּח.**
   *(nim-TSÁ ba-mat-BÁ-akh)*
   "Je me trouve dans la cuisine."

5. **(Passé, tu - féminin)** נִמְצֵאת **בָּרְחוֹב.**
   *(nim-TSÉT ba-re-KHÓV)*
   "Tu te trouvais dans la rue."

### Corrigé Exercice 5

1. **הַבַּיִת נִמְצָא בֵּין שְׁנֵי רְחוֹבוֹת.**
