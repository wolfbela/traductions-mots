# data scientist - מַדְעַן נְתוּנִים (mada'an netunim)

**Genre :** Masculin

## Exemples

**עִבְרִית :** הִיא מַדְעֲנִית נְתוּנִים
**Phonétique :** Hi mada'anit netunim
**Français :** Elle est data scientist

## Mots liés

- נְתוּנִים (netunim) - données
- מַדָּע (mada) - science
- מַדְעָן (mada'an) - scientifique

## Carte Anki

**RECTO :** data scientist

**VERSO :** מַדְעַן נְתוּנִים (m.) / מַדְעֲנִית נְתוּנִים (f.)
(mada'an netunim / mada'anit netunim)
